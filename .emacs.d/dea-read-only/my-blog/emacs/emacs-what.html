什么是emacs呢?

让我们先来看看emacs的官网上怎么说:

GNU Emacs is an extensible, customizable text editor—and more.(emacs是一个可扩展,可定制的文本编辑器,但是远远不止于此)

我们再来看看别的人怎么说

王垠:

据我所知，Emacs 的能力包括:

  1. 编辑文本
  2. 编辑程序
  3. 作为其它程序的界面
  4. 作为操作系统
  5. 煮咖啡

大家可以去Google一下更多的网上评论,我就不多举例子了.

我觉得: emacs是一个具有超强扩展性,超强定制性的操作平台. 当你不满意emacs时, 你可以任意的扩展和定制它. 我经常在写程序的时候, 发现不满意的地方, 就停下来定制它. 总之，emacs是“只有你想不到的，没有我做不到的。” 在这里你可以看看emacs长啥样.
